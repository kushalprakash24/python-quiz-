# Python Quiz Application

A simple and interactive command-line-based quiz application built with Python. This project is designed for users to test their knowledge on various topics through a series of multiple-choice questions.

## Features

- Multiple Question Categories:** Includes different topics such as science, math, history, etc.
- Randomized Questions:** Ensures that the order of questions is shuffled.
- Score Tracking:** Keeps track of the user's score.
- User-Friendly Interface:** Simple and easy-to-use command-line interface.
- Customizable Questions:** Add or modify questions as per your requirement.
